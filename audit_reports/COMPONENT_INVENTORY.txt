## Detailed Component Analysis

### Sidebar & Navbar layouts
These layout shells are the most mature elements, using a global CSS grid layout in `main.html` and utilizing standard sidebar collapse logic. However, because 48 templates don't inherit from the layout parent, the sidebar sidebar menu includes are repeated manually, reducing modularity.

### Form Fields and Input Blocks
Form inputs are styled using Bootstrap's `.form-control` and `.form-select`. However, spacing wrapper groups (usually `.mb-3` or custom `.col-lg-6` sizes) are scattered. Forms are styled with manual bootstrap grids rather than modular row abstractions. Form validation classes are mixed.

### Modals and Popups
Bootstrap modals (e.g., `#deleteModal`, `#importModal`) are written inline at the bottom of templates. These could be unified into a single global layout modal structure triggered via data attributes to prevent layout bloating.
